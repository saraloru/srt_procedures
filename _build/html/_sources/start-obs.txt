Start the observations
===========

$ : commands to insert in a shell
> : commands to insert in the operatorInput panel


#. Insert your project number

    ``- project=[projectID]``

#. Initial setup

    ``- antennaReset``

    ``- setupCCB``


#. Select the active surface shape (Shaped configuration for C-band observations)

    ``- asSetup=S``
