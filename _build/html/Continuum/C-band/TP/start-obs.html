Start the observationsÂ¶

$ : commands to insert in a shell

> : commands to insert in the operatorInput panel

: check on the monitor

  1. Insert your project number

    > project=[projectID] Fig.12

  2. Initial setup

    > antennaReset

    > setupCCB Fig.11

  3. Select the active surface shape (Shaped configuration for C-band observations)

    > asSetup=S Fig.2

  4. Insert the Local Oscillator value in MHz

    > setLO=[freq] Fig.11

  5. Select the Total Power backend

    > chooseBackend=BACKENDS/TotalPower Fig.12

  6. Insert the bandwidth (300, 730, 1250 or 2000 MHz) and choose the sample rate (in MHz) :

    > setSection=0,*,[bw],*,*,[sampleRate],* Fig.8

    > setSection=1,*,[bw],*,*,[sampleRate],*

  7. Put the antenna at 45 deg of elevation and attenuate the signal in order to obtain values between 750 and 1100 counts (linear range of the backend) :

    > goTo=*,45d Fig.9

    > getTpi

    > setAttenuation=0,[att] with [att] between 0 and 15 dB Fig.8

    > setAttenuation=1,[att] with [att] between 0 and 15 dB

    > getTpi

  8. Check the tsys (typical values 35,29 K)

    > tsys Fig.8

  9. Report the ground temperature, relative humidity, atmospheric pressure, and wind speed :

    > wx

  10. Follow the link below to perform the pointing and focus optimization (if not already included in your schedule) :

    Pointing and Focus optimization

  11. Open the calibrationtool client in a terminal to display the pointing and focus plots in real time :

    $ calibrationtoolclient MANAGEMENT/CalibrationTool Fig.14

  12. Begin the schedule by indicating the start scan [N] or subscan

[N_n] in the .scd file :

> startSchedule=[projectID]/[schedulename].scd,[N] Fig.12
