Start the observationsÂ¶

$ : commands to insert in a shell

> : commands to insert in the operatorInput panel

: check on the monitor

  1. Insert your project number :

    > project=[projectID] Fig.12

  2. Initial setup :

    > antennaReset

    > setupCCB Fig.11

  3. Select the active surface shape (Shaped configuration for C-band observations) :

    > asSetup=S Fig.2

  4. Insert the Local Oscillator value in MHz :

    > setLO=[freq] Fig.11

  5. Select and configure the XARCOS backend in C-band :

    > chooseBackend=BACKENDS/XBackends Fig.12

    $ genericBackendTui BACKENDS/XBackends Fig.15

    > initialize=XC00

  6. The initialize command, which is also inserted in the schedule (.bck) directly, sets all XARCOS parameters such as frequency, bandwidth and sample rate. You can check that the backend parameters are correct in the BACKENDS/XBackends Tui, or modify them by using, i.e., the following command:

    > setSection=[sect],[startFreq],[bw],*,*, [sampleRate],*

    with :

    • [sect]: 0 in full-Stokes observations ;
    • [startFreq] corresponds to the initial frequency in the 125-250 MHz range from the LO value ;
    • [bw] the bandwidth : 125.0, 62.5, 31.25, 15.625, 7.8125, 3.90625, 1.953125, 0.9765625 or 0.48828125 MHz ;
    • [sampleRate]: in MHz, must be twice the bandwidth.
    • * : indicates the number of feeds, polarization mode and frequency channels, respectively. Let the asterix (*) for Xarcos observations.

  7. Report the ground temperature, relative humidity, atmospheric pressure, and wind speed :

    > wx

  8. To perform the pointing and focus optimization (if they are not already included in your schedule), use the Total Power backend and follow the link below:

    > chooseBackend=BACKENDS/TotalPower

    Pointing and Focus optimization

  9. Begin the schedule by indicating the start scan [N] or subscan [N_n] in the SCD file :

    > startSchedule=[projectID]/[schedulename].scd,[N] Fig.12
